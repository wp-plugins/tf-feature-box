== Description ==

This plugin adds a feature box with a title, description, coloured background, icon and main image.

1. Add the widget to your sidebars.
1. Use multiple widgets in the same sidebar.
1. Customize the output through your own CSS.

== Installation ==

1. Unzip and upload `rf-feature-box` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
